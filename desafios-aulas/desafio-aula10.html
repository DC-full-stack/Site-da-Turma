Desafio aula 10

Agora que você já viu alguns conceitos básicos, é hora de praticar.

Para um melhor aprendizado, praticar é fundamental, pois não basta só entender os conceitos e não ver a interação acontecendo conforme você escreve o seu código.

Vamos ao desafio.

Na área de trabalho abra o git bash e usando os comandos que você viu tente seguir o seguinte caminho:
Crie o diretório "aula10-git" > entre na pasta "aula10-git" > crie arquivo "git.html" e abra o seu editor de código (se for o VS Code abra usando o git bash)

Agora vamos iniciar um repositório local e linkar ele com o github, seu repositório remoto.

Dica:

Leia as informações do seu repositório remoto, elas são bem explicativas.

Com o repositório local criado e linkado ao repositório remoto, tente adicionar o arquivlo git.html ao repositório remoto

Dica:

git add ____ ...

Boa sorte e bons estudos!
