Desafio aula 13

Agora que você já viu alguns conceitos básicos, é hora de praticar, abra o editor de código de sua preferência e mãos na massa.

Para um melhor aprendizado, praticar é fundamental, pois não basta só entender os conceitos e não ver a interação acontecendo conforme você escreve o seu código.

Vamos ao desafio.

  1. Abra o git bash e troque o nome da branch master para main
  2. Crie uma nova branch e então escreva os componentes necessários para inicializar uma página HTML e escreva no body a mensagem "Hello World"
  3. Faça o git push para um repositório remoto
  4. Por fim faça o merge da nova branch que você criou com a main

Boa sorte e bons estudos!
