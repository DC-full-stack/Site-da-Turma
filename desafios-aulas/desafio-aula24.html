Desafio aula 24

Agora que você já viu alguns conceitos básicos, é hora de praticar, abra o editor de código de sua preferência e mãos na massa.

Para um melhor aprendizado, praticar é fundamental, pois não basta só entender os conceitos e não ver a interação acontecendo conforme você escreve o seu código.

Vamos ao desafio.

Crie um arquivo HTML que contenha uma div com a classe ".botao" com a palavra botão e fundo verde, capture esse elemento com o js e faça os seguintes eventos:

  • Quando o mouse entrar no botão deve aparecer "entrou"
  • Quando o mouse sair no botão deve aparecer "saiu"
  • Quando clicar deve aparecer a palavra "Clicou" e o fundo do botão deve mudar de cor para vermelho
  • Quando clicar novamente faça o fundo mudar novamente para verde

Dica:

Pesquise .addEventListener na documentação da MDN

Boa sorte e bons estudos!
