Desafio aula 2

Agora que você já viu alguns conceitos básicos, é hora de praticar, instale o editor de código de sua preferência e mãos na massa.

Sugestões de editores de códigos: Visual studio code e Sublime Text

Para um melhor aprendizado, praticar é fundamental, pois não basta só entender os conceitos e não ver a interação acontecendo conforme você escreve o seu código.

Vamos ao desafio.

Com o editor de códigos já aberto, escreva os componentes necessários para inicializar uma página HTML. E escreva no body a mensagem "Hello World".

Dica:

        <!DOCTYPE html>
        <html lang=...

Boa sorte e bons estudos!
