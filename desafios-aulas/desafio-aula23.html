Desafio aula 23

Agora que você já viu alguns conceitos básicos, é hora de praticar, abra o editor de código de sua preferência e mãos na massa.

Para um melhor aprendizado, praticar é fundamental, pois não basta só entender os conceitos e não ver a interação acontecendo conforme você escreve o seu código.

Vamos ao desafio.

  • Imprima os elementos do Array seguinte usando o console.log(' ')
    let arr=[1,3,4,10,20].
  • Tente substituir os valores 1, 4 e 20 do array anterior por 0, 9, 15.
  • Usando a estrutura de repetição for imprima o nome dos alunos do seguinte Array
    let alunos=['joao','pedro','joaquim'].
  • Crie um função de soma e faça ela retorar a soma dos valores, considerando as seguintes variáveis:
    let el = soma (10);
    let d = soma (el);
    let y = 2 (y deve ser uma variável local);
    let z = 3.

Boa sorte e bons estudos!
