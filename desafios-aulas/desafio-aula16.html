Desafio aula 16

Agora que você já viu alguns conceitos básicos, é hora de praticar, abra o editor de código de sua preferência e mãos na massa.

Para um melhor aprendizado, praticar é fundamental, pois não basta só entender os conceitos e não ver a interação acontecendo conforme você escreve o seu código.

Vamos aos desafios.

  1. Crie notas de um aluno e as guarde dentro de 3 variáveis.
    Em uma quarta variável você deve usar os operadores para calcular a média das notas.
    Agora monte uma estrutura de condição para as seguintes informações:

    • Se o aluno conseguir uma média igual ou maior a 7, imprima a mensagem 'aprovado'
    • Se o aluno conseguir uma média de 5 a 7, imprima a mensagem 'af'
    • Se o aluno conseguir uma média inferior a 5, imprima a mensagem 'reprovado'

  2. Usando o console do DevTools você deve usar o comando console.log(' ') para escrever 5 tipos de dados.

Boa sorte e bons estudos!
