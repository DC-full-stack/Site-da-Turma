Desafio aula 12

Agora que você já viu alguns conceitos básicos, é hora de praticar, abra o git bash e mãos na massa.

Para um melhor aprendizado, praticar é fundamental, pois não basta só entender os conceitos e não ver a interação acontecendo conforme você escreve o seu código.

Vamos ao desafio.

No Github escolha um repositório remoto de sua preferência e faça um clone no seu aparelho.
Com clone feito, crie uma nova branch e faça algumas alterações.
Tente navegar entre as branchs até ficar confortável com o comando.
Depois tente apagar alguma branch.

Boa sorte e bons estudos!
