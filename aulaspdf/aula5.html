ID e Class

  • “O atributo Id especifica uma identificação única para o elemento HTML.”
    • Localiza com "#"
  • “O atributo global class especifica uma ou mais classes para o elemento HTML.”
    • Localiza com "."

Seletor {}

  • Seletor seleciona elementos dentro do CSS e diz as características;
  • Seletor universal * {}
  • Quando mais de uma CSS atinge um elemento, o navegador dar prioridade à CSS mais específica;
  • Comentário em CSS, JavaScript e PHP: /* */
  • Comentário em HTML: <!–- -->
